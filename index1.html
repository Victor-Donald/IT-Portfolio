Project 2: Enterprise Network Diagnostics & Infrastructure Support Simulation

DeVry University Campus | October 2010 - December 2012

🛠️ Technical Execution

Fielded a critical help request regarding a total loss of intranet and database access. Utilized advanced command-line interface (CLI) diagnostics (ipconfig /all) to identify an autoconfigured APIPA address (169.254.x.x). Coordinated an immediate switch port reset with infrastructure teams and executed an IP release/renew cycle to bind the client device to a valid dynamic subnet.

📈 Business Value & Outcome

Restored critical administrative and student registration workflows, preventing extended operational disruptions. Confirmed physical NIC functionality via loopback testing and verified secure database access while successfully meeting all SLA compliance metrics.
